# PRD - 测测你的人性弱点 (Weak)

> 本文档定义「测测你的人性弱点」产品的完整产品需求，基于卡内基《人性的弱点》设计。

---

## 一、产品概述

### 1.1 产品定位

**一句话描述：** 一款基于卡内基《人性的弱点》的AI性格测试工具，通过25道选择题诊断你在人际交往中的"致命弱点"。

**核心用户画像：**
- 主要用户：18-35岁职场年轻人（刚入职/转行/想提升情商）
- 次要用户：对人际交往困惑的学生、自由职业者
- 用户特征：想知道自己在社交中的问题、愿意为"自我认知"付费

### 1.2 核心价值主张

> "25道题，测出你在人际交往中的致命弱点"

用户购买的不是一个标签，而是"知道自己哪里做错了"和"如何改正"的具体路径。

### 1.3 灵感来源：卡内基《人性的弱点》

卡内基把人际交往的弱点归纳为**六大维度**：

| 维度 | 卡内基原文核心 | 测试问法 |
|------|--------------|---------|
| **批评型** | "不要批评、指责或抱怨" | 你在关系中是否倾向于挑剔和批评他人？ |
| **冷漠型** | "真诚地关注他人" | 你是否只在乎自己的需求，忽视对方的感受？ |
| **自恋型** | "谈论对方感兴趣的话题" | 聊天时你是否只说自己，不问对方的事？ |
| **记不住型** | "记住对方的名字" | 你是否经常忘记别人的名字和基本信息？ |
| **防御型** | "在争论中获胜的唯一方法是避免争论" | 遇到分歧时你是否必须赢？ |
| **索取型** | "激发他人内在需求" | 你是否只关注自己想从对方那里得到什么？ |

### 1.4 商业模式

```
小红书发布内容 → 引流到测试网站
        ↓
用户访问 → 输入周码（¥0.99/周码）
        ↓
做25道题 → 获得弱点诊断报告
        ↓
分享到小红书 → 带来二次传播
```

## 三、结果页设计（已实现）

### 3.1 页面结构

1. **维度得分雷达** - 6维度得分可视化条形图
2. **主要弱点卡片** - icon + 名称 + 卡内基原文引言
3. **为什么形成这个弱点** - 心理分析
4. **日常生活表现** - 3条具体行为描述
5. **卡内基改变建议** - 3条可执行的具体行动
6. **次要弱点提示** - 第二高得分维度
7. **书籍推荐** - 《人性的弱点》卡内基
8. **核心忠告** - 一句话总结
9. **分享按钮** - 复制文案到剪贴板

### 3.2 设计风格

- 配色：深色背景(#0f0f1a → #1a1a2e) + 红色警示(#e94560)
- 字体：Noto Sans SC
- 风格：简洁有力，突出"被戳中"的感觉

## 五、题库规格

- 总题数：25道
- 每道题：4个选项
- 每选项分值：-1 / 0 / 1 / 2
- 每维度：4题，满分8分
- 干扰题：1题（Q22，不计入维度）
